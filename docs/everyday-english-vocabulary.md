# Everyday English Words Japanese Speakers Don't Know

日本で生活していると絶対に出会わない「英語圏の日常単語」15,000語リスト。
TOEIC 900点でも、英検1級でも、日本にいる限り知らない可能性が高い単語たち。

> varsity, detergent, pantry... こういう「向こうでは毎日使うけど日本の教科書には出ない」単語を集めました。

## How to Use This List
- 各カテゴリは生活シーンごとに分類
- 「英語」「読み方ヒント」「意味」「補足」の4列
- 星マーク付き = 特に知らない人が多い重要単語
