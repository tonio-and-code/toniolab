# Journal Data Structure

## 📁 Directory Structure

```
src/data/
├── journal.ts          # 既存の全エントリ（そのまま維持）
└── journal/            # 新しい構造（2026年以降）
    ├── types.ts        # 型定義
    ├── index.ts        # 全エントリを統合
    ├── 2026/
    │   └── 01-january.ts
    └── 2025/
        └── (今後追加)
```

## 📝 使い方

### 新しいエントリを追加する場合

**オプション1: 既存の方法（推奨・簡単）**
→ `src/data/journal.ts` の先頭に追加

**オプション2: 新しい構造**
→ 該当する月のファイル（例：`journal/2026/01-january.ts`）に追加

### 検索する場合

1. エントリのIDや日付で検索
2. 該当する年月のファイルを開く
3. ファイルが小さいので探しやすい

## ⚠️ 重要

- `journal.ts` は削除しないこと
- 新旧両方の構造が共存できます
- 徐々に移行する予定

## 🎯 今後の方針

- 2026年以降の新しいエントリは月別ファイルに追加
- 過去のエントリは `journal.ts` のまま
- 必要に応じて徐々に移行
